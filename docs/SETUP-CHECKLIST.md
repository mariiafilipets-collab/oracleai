# Чек-лист настройки: Claude Code + агенты в Cursor

По шагам — что выбрать и что проверить.

---

## 1. Окно с кнопками «Undo All» / «Keep All» / «Review»

Если Cursor показывает изменения в файлах (доки, правила):

- **Выбери: «Keep All»** — принять все правки (документация, `.cursor/rules/`, `CLAUDE.md` и т.д.). Так у тебя будут актуальные гайды и правила для агентов.
- Если хочешь проверить по файлам — нажми **«Review»**, посмотри дифф, потом прими нужное (или снова «Keep All»).

**Итог:** нажми **Keep All**.

---

## 2. PATH для Claude Code (если ещё не делал)

- Открой **новое** окно PowerShell (или перезапусти терминал в Cursor).
- Выполни:
  ```powershell
  claude --version
  ```
- Если команда не найдена — добавь в PATH путь `C:\Users\user\.local\bin` (инструкция в `docs/CLAUDE-CODE-PATH.md`), закрой и снова открой терминал, проверь ещё раз.

**Итог:** в новом терминале должна выполняться команда `claude --version`.

---

## 3. Первый вход в Claude Code

В том же (или новом) терминале:

```powershell
cd D:\OAI
claude
```

- Откроется браузер для входа в Anthropic.
- Войди (нужна подписка Pro или API-кредиты).
- После входа можно в том же терминале писать запросы к Claude; контекст подтянется из `CLAUDE.md` в корне проекта.

**Итог:** один раз залогинился и убедился, что `claude` в терминале отвечает.

---

## 4. Агенты в Cursor (панель «Agents»)

- **«New Agent»** — создаёт новую сессию. Удобно завести отдельные сессии, например:
  - «Разработка фич» (backend + frontend),
  - «Тесты и QA»,
  - «Контракты».
- В чате с агентом можно писать: *«Действуй как FullstackEngineer»* или *«Действуй как QAEngineer»* — тогда ориентируйся на `ops/agents/fullstack-engineer.system.md` и `ops/agents/qa-engineer.system.md`.
- Правила в `.cursor/rules/` подхватятся автоматически при открытии файлов (backend, frontend, contracts, тесты).

**Итог:** создай при необходимости 1–2 агента под задачи, в запросах при желании указывай роль.

---

## 5. Быстрая проверка

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 1 | В окне с изменениями нажать **Keep All** | Файлы сохранены, окно закрылось |
| 2 | Новый терминал → `claude --version` | Версия (например 2.1.74) |
| 3 | `cd D:\OAI` → `claude` | Браузер, вход, ответ в терминале |
| 4 | В Cursor открыть файл из `backend/` или `frontend/` | Правила агента подхвачены (видно по контексту в чате) |

---

## Если что-то не работает

- **`claude` не найден** → `docs/CLAUDE-CODE-PATH.md`.
- **Установка и роли агентов** → `docs/CLAUDE-CODE-SETUP.md`.
- **Контекст для Claude** → корневой `CLAUDE.md`.

После прохождения чек-листа всё будет настроено: и Claude Code в терминале, и несколько агентов в Cursor.
